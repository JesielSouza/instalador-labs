# 🎯 Visão Geral do Projeto

## Objetivo
[cite_start]Automatizar a padronização de softwares nos laboratórios do campus utilizando Python e WinGet, reduzindo o esforço manual e erros operacionais[cite: 3].

## [cite_start]Problemas Resolvidos [cite: 6]
* [cite_start]Instalação manual repetitiva. [cite: 6]
* [cite_start]Inconsistência entre máquinas do mesmo laboratório. [cite: 6]
* [cite_start]Falta de rastreabilidade (quem instalou o quê e quando). [cite: 6]

## [cite_start]Stack Tecnológica [cite: 7]
* Python 3.10+
* WinGet (Windows Package Manager)
* JSON (Base de dados de pacotes)
